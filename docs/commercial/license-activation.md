# License Activation

There is no license activation.

`plsql-intelligence` is fully open source under `Apache-2.0 OR MIT`.
There is no license key, no activation server, no commercial tier, and no
separate gated tool surface.

The current repository ships the offline PL/SQL analysis engine as Rust
crates and CLIs. MCP transport and live database operation are handled by
`oraclemcp`.
