# MCP Directory Listing Ownership

MCP directory submissions belong to `oraclemcp`.

This repository should be described as the offline PL/SQL intelligence
engine that powers deeper Oracle analysis. It is not the runnable MCP
server listing target.

Use this positioning when linking the project from `oraclemcp` docs or
directory metadata:

```text
Offline Oracle PL/SQL code intelligence engine: parser, catalog snapshot
model, dependency graph, lineage, SAST, change-impact, docs, bindings, and
CI/CD prediction for Oracle PL/SQL estates.
```

Directory checklists for agent transport, live database credentials,
container images, registry entries, and server health checks should live in
the `oraclemcp` repository.
